### Quality-Quiz: Performance Testing (5 questions)

1️⃣ *What does 'p95 latency' typically refer to?*
   - A) The 95th percentile response time — 95% of requests are faster
   - B) The 95% slowest requests
   - C) The average latency
   - D) The minimum latency
**Answer:** The 95th percentile response time — 95% of requests are faster
**Explanation:** p95 is a common SLO metric that captures the experience of most users while ignoring extreme outliers.

2️⃣ *Stress testing verifies how a system behaves beyond normal load conditions.*
**Answer:** True
**Explanation:** Stress testing pushes the system past expected capacity to identify breaking points and recovery behavior.

3️⃣ *Which of the following are common performance bottlenecks in web applications? (Select all that apply)*
   - A) Unindexed database queries
   - B) Excessive synchronous I/O
   - C) Connection pool exhaustion
   - D) Efficient caching
   - E) Large payload sizes
**Answer:** Unindexed database queries, Excessive synchronous I/O, Connection pool exhaustion, Large payload sizes
**Explanation:** Slow queries, blocking I/O, exhausted pools, and bloated payloads all degrade performance. Efficient caching is a solution, not a bottleneck.

4️⃣ *Which of the following are types of performance testing? (Select all that apply)*
   - A) Load testing
   - B) Stress testing
   - C) Soak testing
   - D) Spike testing
   - E) Smoke testing only
**Answer:** Load testing, Stress testing, Soak testing, Spike testing
**Explanation:** Load, stress, soak, and spike tests all measure system behavior under different load patterns. Smoke testing is a functional check, not a performance category.

5️⃣ *What is the primary goal of load testing?*
   - A) To find security bugs
   - B) To measure system behavior under expected user load
   - C) To validate UI design
   - D) To replace unit tests
**Answer:** To measure system behavior under expected user load
**Explanation:** Load testing simulates expected traffic to verify response times, throughput, and stability.
